# TWiR Submission

## How to submit

1. Fork https://github.com/this-week-in-rust/this-week-in-rust
2. Find the latest `draft/YYYY-MM-DD-this-week-in-rust.md` file
3. Add this line under `### Project/Tooling Updates`:

```markdown
* [axum-harness: agent-native backend architecture template for Axum — semantic-first, topology-late, multi-agent harness](https://github.com/openclosed-org/axum-harness/blob/main/docs/launch.md)
```

4. Open a PR
