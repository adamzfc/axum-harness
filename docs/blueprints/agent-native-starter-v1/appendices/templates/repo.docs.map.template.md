# 仓库文档地图模板

```text
docs/
  README.md
  blueprints/
    agent-native-starter-v1/
  architecture/
  conventions/
  testing/
  release/
  agents/
  playbooks/
  adr/
  examples/
  anti-patterns/
```

## 每类文档职责

- `blueprints/`：长期蓝图与制度层
- `architecture/`：当前仓库实现结构
- `conventions/`：命名、目录、规则
- `testing/`：测试与 eval 体系
- `release/`：发布与回滚
- `agents/`：agent 操作说明
- `playbooks/`：任务指南
- `adr/`：关键架构决策记录
- `examples/`：正确做法
- `anti-patterns/`：禁止做法
```
