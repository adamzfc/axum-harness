# Key note: Host-specific vs Business logic
Auth commands (start_oauth, handle_oauth_callback, get_session) use Tauri plugins directly (plugin-store, plugin-opener). They are host-specific adapter code, NOT pure business logic. This is correct — adapters translate host APIs into usecase calls.
