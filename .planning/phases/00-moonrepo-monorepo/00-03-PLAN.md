    ```json
    {
      "$schema": "https://raw.githubusercontent.com/nicoverbruggen/tauri-settings/main/schema.json",
      "productName": "geek taste",
      "version": "0.1.0",
      "identifier": "com.geek-taste.app",
      "build": {
        "beforeDevCommand": "pnpm dev",
        "devUrl": "http://localhost:5173",
        "beforeBuildCommand": "pnpm build",
        "frontendDist": "../build"
      },
      "app": {
        "windows": [
          {
            "title": "geek taste",
            "width": 1200,
            "height": 800,
            "resizable": true,
            "fullscreen": false
          }
        ],
        "security": {
          "csp": null
        }
      },
      "bundle": {
        "active": true,
        "targets": "all",
        "icon": [
          "icons/32x32.png",
          "icons/128x128.png",
          "icons/128x128@2x.png",
          "icons/icon.icns",
          "icons/icon.ico"
        ]
      }
    }
    ```

    2. `Cargo.toml`（src-tauri 自身的，也作为 workspace member）:
    ```toml
    [package]
    name = "desktop-ui-tauri"
    version = "0.1.0"
    edition = "2021"

    [build-dependencies]
    tauri-build = { workspace = true }

    [dependencies]
    tauri = { workspace = true }
    tauri-plugin-shell = { workspace = true }
    tauri-plugin-dialog = { workspace = true }
    tauri-plugin-store = { workspace = true }
    tauri-plugin-notification = { workspace = true }
    serde = { workspace = true }
    serde_json = { workspace = true }
    tracing = { workspace = true }
    ```

    3. `build.rs`:
    ```rust
    fn main() {
        tauri_build::build()
    }
    ```

    4. `src/lib.rs`:
    ```rust
    //! desktop-ui-tauri — Tauri application entry point

    #[cfg_attr(mobile, tauri::mobile_entry_point)]
    pub fn run() {
        tauri::Builder::default()
            .plugin(tauri_plugin_shell::init())
            .plugin(tauri_plugin_dialog::init())
            .plugin(tauri_plugin_store::Builder::default().build())
            .plugin(tauri_plugin_notification::init())
            .run(tauri::generate_context!())
            .expect("error while running tauri application");
    }
    ```

    **注意:** 根 Cargo.toml workspace members 需要更新以包含 `apps/desktop-ui/src-tauri`。
    更新根 `Cargo.toml` members 为：
    ```toml
    members = [
        "apps/desktop-ui/src-tauri",
        "crates/*",
    ]
    ```
    （移除 `apps/*` glob，因为 apps/desktop-ui 不是 Rust crate，只有 src-tauri 是）

    或者保持 `apps/*` glob 不变，添加 `apps/desktop-ui/src-tauri` 为显式 member：
    ```toml
    members = [
        "apps/*",
        "apps/desktop-ui/src-tauri",
        "crates/*",
    ]
    ```
    因为 `apps/*` 只匹配一级子目录，不匹配 `apps/desktop-ui/src-tauri`。
